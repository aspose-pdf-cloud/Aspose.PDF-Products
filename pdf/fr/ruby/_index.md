<p>Aspose.PDF Cloud Ruby SDK vous aide à construire des applications Ruby basées sur le cloud pour générer des documents PDF sur le stockage cloud et améliorer votre code Ruby pour travailler avec des pages PDF cloud, des champs de formulaire, des éléments de texte, des images, des annotations (ligne, cercle, carré, souligné, ondulé), des signets, des liens et des pièces jointes via un SDK cloud Ruby facile à utiliser.</p>
<p>Il permet à vos programmes Ruby de créer des fichiers PDF basés sur le cloud à partir de différents formats, tels que HTML, XML, PCL, SVG, XPS, JPEG, TIFF, et de convertir également des fichiers PDF Acrobat distants en formats TIFF, XPS, DOC, DOCX, sans installer de logiciel tiers.</p>
<p>Vous pouvez diviser de plus gros fichiers PDF en plusieurs fichiers et les enregistrer tous comme des documents cloud séparés en utilisant des commandes Ruby simples. De même, Aspose.PDF Cloud Ruby SDK vous permet de fusionner des documents PDF cloud, crypter, décrypter, ajouter, importer et exporter.</p>

<div class="col-lg-12">
<h2 class="h2title">Commencez avec le SDK de traitement de documents PDF</h2>
<p>Il est facile de commencer avec Aspose.PDF Cloud Ruby SDK et il n'y a rien à installer. Créez simplement un compte sur<a href="https://dashboard.aspose.cloud/#/apps">Aspose for Cloud</a>et obtenez les informations de votre application. Une fois que vous avez le SID de l'application &amp; la clé, vous êtes prêt à utiliser Aspose.PDF Cloud Ruby SDK.</p>
</div>
<div class="col-lg-12">
<h2 class="h2title">Fonctionnalités étendues pour la conversion PDF</h2>
<p>Aspose.PDF Cloud Ruby SDK vous permet de convertir vers et depuis PDF vers divers autres formats de fichiers populaires. Avec celui-ci, vous pouvez convertir EPUB, HTML, LaTeX, MHT, PCL, DOC, MOBIXML, PDFA, PPTX, SVG, TIFF, XLS, XLSX, XML, XPS, PS, XSLFo, PCL, BMP, EMF, GIF, JPEG, PNG et TIFF, en PDF.</p>
<p>Vous pouvez également convertir PDF (avec formulaires XFA) en PDF avec AcroForm. De même, Aspose.PDF Cloud Ruby SDK vous permet de convertir PDF en BMP, PDF EMF, GIF, JPEG, PNG, TIFF, DOC, EPUB, HTML, LaTex, MOBIXML, PDFA, PPTX, SVG, XLS, XLSX, XML, et XPS.</p>
<p>De plus, Aspose.PDF Cloud Ruby SDK vous permet de contrôler comment un document PDF est converti en document de traitement de texte. Vous pouvez choisir entre le mode “Textbox” pour préserver l'apparence originale du fichier PDF ou le mode “Flow” pour une analyse multi-niveaux avec un accent sur l'édition maximale.</p>
<p>Vous pouvez également définir des privilèges d'accès au document pour les fichiers PDF. Ces privilèges incluent l'impression, l'impression dégradée, la modification du contenu, la copie du contenu, le remplissage de formulaires PDF et bien d'autres.</p>
</div>
<div class="col-lg-12">
<h2 class="h2title">Annotations personnalisées</h2>
<p>Aspose.PDF Cloud Ruby SDK prend en charge divers types d'annotations, telles que l'annotation de texte, l'annotation de cercle, l'annotation de polygone, l'annotation de polyligne, l'annotation de ligne, l'annotation de carré, l'annotation de texte libre, l'annotation de surlignement, l'annotation de soulignement, l'annotation de vague, l'annotation de rature, l'annotation de crochet, l'annotation d'encre, l'annotation de lien, l'annotation contextuelle, l'annotation de pièce jointe, l'annotation sonore, l'annotation de film, l'annotation d'écran, l'annotation de widget, l'annotation de filigrane, l'annotation de réseau de piège (TrapNet), l'annotation de marqueur d'imprimante, l'annotation de caviardage, l'annotation de tampon en caoutchouc et l'annotation multimédia enrichie.</p>
<p>Vous pouvez définir des indicateurs pour spécifier diverses caractéristiques des annotations prises en charge. Définissez la visibilité, la restriction d'impression, la restriction de zoom, la restriction de rotation, rendez les annotations en lecture seule et d'autres indicateurs.</p>
<p>Avec Aspose.PDF Cloud Ruby SDK, les annotations peuvent être définies à divers états tels que marqué, non marqué, accepté, rejeté, annulé, complété ou aucun. Il vous permet également de définir diverses propriétés de tampons pour les documents PDF. Ces propriétés incluent l'opacité, la rotation, l'alignement, l'indentation, le zoom et diverses autres propriétés.</p>
</div>
<div class="col-lg-12">
<h2 class="h2title">Extraire divers éléments du document PDF</h2>
<p>Aspose.PDF Cloud Ruby SDK vous permet d'extraire divers éléments des documents PDF. Vous pouvez extraire des images du document PDF au format GIF, JPEG, PNG, TIFF et les enregistrer dans un dossier. Il vous permet également d'extraire les annotations du document PDF, telles que l'annotation de pièce jointe, l'annotation d'écran, l'annotation sonore et l'annotation de tampon dans un dossier spécifié.</p>
</div>
<div class="col-lg-12">
<h2 class="h2title">Optimiser les documents PDF</h2>
<p>Aspose.PDF Cloud Ruby SDK vous permet de définir diverses propriétés pour optimiser les documents PDF. Il vous permet d'optimiser la taille du fichier PDF en permettant la réutilisation du contenu de la page, de définir le niveau de compression des images intégrées dans le document PDF, de lier les flux de ressources en double en les stockant comme un seul objet, de supprimer les objets du document sans aucune référence, de supprimer les flux inutilisés et de ne pas intégrer les polices du document PDF.</p>
</div>
