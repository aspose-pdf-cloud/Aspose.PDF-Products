```js

    async function () {
        const pdfApi = new PdfApi(credentials.id, credentials.key);
        try {
            const fileBuffer = await fs.readFile(LOCAL_FILE_NAME);
            await pdfApi.uploadFile(STORAGE_FILENAME, fileBuffer);
            const result = await pdfApi.getText(STORAGE_FILENAME, 0, 0, 0, 0);
            const lines = result.body.textOccurrences.list.map(line=>line.text).join("\n");
            await fs.writeFile("extracted.txt", lines);
        }
        catch (error) {
            console.error(error.message);
        }
    }
```

L'extraction de texte permet d'analyser, d'organiser ou de traiter les données des PDF dans des applications externes. Le texte extrait peut être indexé, ce qui le rend consultable à travers des bases de données ou des systèmes de gestion de contenu.
Cela améliore la récupération des documents et permet un accès plus rapide aux informations spécifiques, surtout dans de grandes archives de documents.
