<p>Aspose.PDF Cloud Java SDK dựa trên Aspose.PDF REST API nền tảng và hỗ trợ các nhà phát triển Java tích hợp liền mạch chức năng xử lý và thao tác PDF trên đám mây vào các ứng dụng Java của bạn được xây dựng cho các nền tảng đám mây. Aspose.PDF Cloud Java SDK được cung cấp dưới giấy phép MIT.</p>
<p>Aspose.PDF Cloud Java SDK cho phép các ứng dụng Java dựa trên đám mây của bạn tạo tài liệu PDF từ đầu hoặc chuyển đổi các định dạng khác nhau sang PDF để xử lý thêm. Bạn có thể chuyển đổi các định dạng EPUB, LaTeX, HTML, PS, XsIFo, XPS, SVG, DOCX, PCL, XML và MD (Markdown) sang PDF bằng SDK Java dễ sử dụng. Tương tự, mã Java được nâng cao của bạn có thể chuyển đổi PDF sang nhiều định dạng tệp khác, chẳng hạn như hình ảnh, Word, PowerPoint, Excel và các định dạng khác.</p>
<p>Aspose.PDF Cloud Java SDK hỗ trợ bạn làm việc với các yếu tố riêng lẻ của tài liệu PDF trên đám mây, chẳng hạn như văn bản, chú thích, dấu watermark, chữ ký, tem, dấu trang, trường biểu mẫu, hình ảnh, siêu liên kết, tệp đính kèm, v.v. Bạn cũng có thể tách, ghép, ký và mã hoá các tệp PDF lưu trữ trên đám mây.</p>

<p>Hơn nữa, Aspose.PDF Cloud Java SDK cho phép bạn kiểm soát cách tài liệu PDF được chuyển đổi thành tài liệu xử lý Word. Bạn có thể chọn chế độ “Textbox” để giữ nguyên giao diện gốc của file PDF hoặc chế độ “Flow” cho phân tích đa cấp với mục tiêu có khả năng chỉnh sửa tối đa.</p>
<p>Bạn cũng có thể đặt các quyền truy cập tài liệu cho các file PDF. Các quyền này bao gồm in, in giảm chất lượng, sửa đổi nội dung, sao chép nội dung, điền vào các biểu mẫu PDF và nhiều hơn nữa.</p>

Nó cũng cho phép bạn thiết lập các thuộc tính khác nhau cho con dấu trong tài liệu PDF. Các thuộc tính này bao gồm độ trong suốt, xoay, căn chỉnh, thụt lề, thu phóng và các thuộc tính khác.</p>
